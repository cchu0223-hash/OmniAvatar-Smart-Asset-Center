## 三、AI素材面板

点击侧栏「图片」进入，顶部 Tab 切换：**本地素材 | AI素材**。

### 3.1 统一生成面板（图片 & 视频合一）

图片生成和视频生成合并为一个面板，通过类型切换控制内容。

#### 提示词区

```
┌────────────────────────────────────────┐
│ [REFERENCE 上传参考图]  提示词 textarea  │
└────────────────────────────────────────┘
```

- 左侧：80×96 px 虚线参考图区域，标注「REFERENCE」
- 右侧：多行文本输入，高度自适应
- 无「示例」按钮

#### 参数控制行（单行）

```
[图片|视频] | [比例▾] [模型▾] [分辨率▾] [时长▾(仅视频)]
```

| 控件 | 图片模式 | 视频模式 |
|------|---------|---------|
| 类型切换 | 图片（选中蓝色） | 视频（选中蓝色） |
| 比例 | 16:9 / 9:16 / 4:3 / 1:1 | 同左 |
| 模型 | JIMENG 4.0 | KELING 3.0  |
| 分辨率 | 720p / 1080p / 2K | 720p / 1080p / 2K |
| 时长 | 不显示 | 滑动选择 |

每个下拉点击弹出 Popover，点击外部关闭。

#### CTA 行

```
剩余算力          [⚡ 20]  [立即生图 / 立即生成]
48,510
```

- 左：剩余算力总额（标签 + 数字）
- 右：消耗算力徽章（深色圆角框 + 青色闪电图标 + 数字）+ 生成按钮（蓝色渐变）
- 图片消耗 20 算力，视频消耗 100 算力

#### 素材列表

- 搜索栏（左侧搜索图标，focus 蓝色描边，按 prompt 过滤）
- 筛选 Tab：**图片 | 视频**（无「全部」）
- 3 列网格，图片 1:1，视频 16:9
- 卡片 hover 显示：放大 / 加入轨道 / 下载 / 删除
- 视频卡片紫色标签「视频」，图片卡片「AI生成」

## 四、素材优化面板

### 4.1 触发逻辑

点击时间轴片段 → 侧栏出现「素材优化」入口 + 面板自动打开：
- AI 生成片段 → 默认「素材更换」Tab
- 本地上传片段 → 默认「动画效果」Tab

### 4.2 Tab 结构

```
[ 素材更换 ]  [ 素材编辑 ]  [ 动画效果 ]
```

### 4.3 素材更换 Tab

布局与 AI素材面板完全一致：

**提示词区**
- 左侧显示当前轨道截图作为参考图（本地素材则显示空白上传框）
- 右侧文本输入

**参数控制行（单行，同 AI素材面板）**

```
[图片|视频] | [比例▾] [模型▾] [分辨率▾] [时长▾(仅视频)]
```

- 图片模式：JIMENG 模型
- 视频模式：KELING 模型 + 时长滑动选择

**CTA 行（同 AI素材面板）**

```
剩余算力          [⚡ 20]  [重新生成]
48,510
```

**素材列表（同 AI素材面板）**
- 搜索栏 + 图片 | 视频 筛选
- 3 列网格，显示图片时额外显示「当前」高亮卡（蓝色 2px 边框）

### 4.4 素材编辑 Tab

参考原设计，不做改动。

### 4.5 动画效果 Tab

入场 / 出场动画各 8 种，选中蓝色高亮。

## 五、跨页面素材同步

通过 `smartMaterialsStore.ts` 实现首页素材中心与专业剪辑页双向同步。

| 操作 | 来源 | 效果 |
|------|------|------|
| 首页生成图片并点击「收藏至素材」 | 首页 | 自动出现在专业剪辑素材列表 |
| 首页生成视频任务点击「收藏至素材」 | 首页 | 同上 |
| 专业剪辑点击「立即生图/生成」 | 专业剪辑 | 自动出现在首页素材中心 |

Store 使用发布订阅模式，组件 `useEffect` 订阅变更。
