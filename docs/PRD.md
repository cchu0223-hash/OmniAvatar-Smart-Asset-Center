### 1.3 输入面板

#### 1.3.1 参考图区域

| 交互 | 说明 |
|------|------|
| 默认状态 | 虚线框 + 上传图标 + "Reference" 文字 |
| 已上传状态 | 显示缩略图，左下角 "参考图" 标签 |
| hover 已上传图 | 覆盖半透明遮罩，显示关闭按钮可移除 |
| 来源 | 用户手动上传，或通过推荐卡片"做同款"自动填入 |

#### 1.3.2 文本输入区

| 交互 | 说明 |
|------|------|
| 占位提示 | 跟随当前类别切换（图片/视频有不同提示语） |
| 右上角"示例"按钮 | 点击随机填入一段示例营销文案 |
| 输入区高度 | 固定 144px，不可 resize |

#### 1.3.3 胶囊控制栏

**左侧区域 — 类别切换：**

| 按钮 | 说明 |
|------|------|
| `图片生成` | 切换至图片模式，带 image 图标 |
| `视频生成` | 切换至视频模式，带 videocam 图标 |
| 激活态 | 渐变背景 (cyan→purple) + 白色边框 + 辉光阴影 |
| 非激活态 | 透明背景，灰色文字，hover 变亮 |
| 切换行为 | 切换类别时自动重置：模型→第一项、比例→第一项、分辨率→第一项、时长→默认值 |

**分隔线：** 垂直细线 `h-8, w-1px, white/10`

**右侧区域 — 参数胶囊（点击向上弹出 Popover）：**

| 胶囊 | 图片模式可选值 | 视频模式可选值 |
|------|--------------|--------------|
| 比例 | 1:1, 3:4, 4:3, 16:9, 9:16 | 3:4, 4:3, 16:9, 9:16 |
| 模型 | Flux Pro, DALL-E 3, Midjourney V6 | Seedance 2.0, Vidu 2.0, Veo 3.0 |
| 分辨率 | 720p, 1080p, 2K | 480p, 720p, 1080p, 2K |
| 时长 | ❌ 不显示 | 4s–15s 滑块 (仅视频模式) |

**Popover 交互规则：**
- 点击胶囊按钮 → 向上弹出菜单 (`popup-up` 动画, 200ms ease-out)
- 已选中项显示 `check_circle` 图标 + cyan 高亮色
- 点击选项 → 更新参数 + 关闭 Popover
- 点击 Popover 外部区域 → 关闭
- 时长胶囊弹出的是滑块控件（非列表），显示当前值和范围

**算力 & 生成按钮：**

| 元素 | 说明 |
|------|------|
| 算力展示 | 图片模式 50，视频模式 100，紫色星星图标 + 数字 |
| 生成按钮 | 渐变背景 (cyan→purple)，120px 最小宽度，点击时 scale(0.95) 反馈 |
| 点击行为 | ① 创建任务加入队列 ② 清空输入框和参考图 ③ 自动切换到"我的"Tab ④ 选中新任务 ⑤ 500ms 后开始模拟进度 |
| 图片任务 | 一次生成 4 张图片（4 个不同 seed），预计 15s |
| 视频任务 | 一次生成 1 个视频，预计 30s |

### 1.4 Feed 区域

#### 1.4.1 Tab 栏

| Tab | 说明 |
|-----|------|
| 推荐 | 展示社区/平台推荐作品（mock 数据） |
| 我的 | 展示用户自己的生成任务和作品 |
| 激活态 | 白色文字 + 底部渐变下划线 (cyan→purple) |
| 搜索按钮 | 右侧放大镜图标，点击展开搜索框，再次点击关闭并清空 |
| 活跃提示 | 当有正在生成的任务时，"我的" Tab 右上角显示 cyan 脉冲圆点 |

#### 1.4.2 推荐 Tab

**瀑布流卡片：**

| 元素 | 说明 |
|------|------|
| 布局 | CSS columns 瀑布流：4列(>1024px)、3列(768-1024px)、2列(<768px) |
| 卡片内容 | 缩略图 + 左上角类型标签(图片/视频) + 底部提示词 + 时间戳 |
| hover 效果 | scale(1.02) + 辉光边框阴影 + 底部渐变遮罩显示"做同款"按钮 |
| 视频卡片 | 中央显示播放按钮（半透明圆形） |
| 点击卡片 | 打开 Lightbox 弹窗查看大图/播放视频 |
| "做同款" | 点击后：① 填入参考图 ② 填入提示词 ③ 自动切换对应类别 ④ 滚动到顶部 |

**骨架屏加载：**
- 初始加载 1.5s 延迟显示 12 条
- 骨架卡片使用 `skeleton-pulse` 动画（脉冲透明度变化）
- 4 个骨架卡片，高度交替变化模拟瀑布流

**无限滚动：**
- "加载更多"按钮，每次加载 8 条
- 最多加载到 40 条后隐藏按钮

**空状态：**
- 居中显示 explore 图标 + "暂无推荐内容" 或 "未找到匹配内容"

**搜索：**
- 实时过滤推荐内容，匹配 prompt 文本

#### 1.4.3 我的 Tab

**空状态：** 居中 palette 图标 + "还没有作品" + "在上方输入描述开始创作"

**有作品时布局：** 左右分栏

| 区域 | 说明 |
|------|------|
| 左侧导航列表 | 宽度 224px，最高 600px 可滚动，按时间倒序排列 |
| 右侧作品详情 | 弹性宽度，最高 600px 可滚动，按 `createdAt` 倒序排列 |

**左侧导航项：**

| 元素 | 说明 |
|------|------|
| 缩略图 | 40×40 圆角方形，生成中显示环形进度条 |
| 文字 | 提示词（单行截断） + 状态标签 + 类型标签 |
| 状态 | 已完成(绿色)、生成中(cyan+百分比)、失败(红色)、排队中(灰色) |
| 选中态 | 白色背景高亮 + cyan 边框辉光 |
| 点击 | 选中该作品 |

**右侧作品卡片 — 图片作品（已完成）：**

| 元素 | 说明 |
|------|------|
| 图片展示 | 4 张图片一行四列 (`grid-cols-4`)，方形比例，hover 微放大 |
| 提示词 | 图片下方，最多 2 行截断 |
| 参数标签 | 胶囊式展示：类型(图片)、模型、比例、分辨率 |
| 提交时间 | 右对齐，格式 `YYYY-MM-DD HH:mm` |
| 操作按钮 | `重新编辑` / `再次生成` / `收藏为我的素材` |
| 选中态 | cyan 边框 + 辉光阴影 |

**右侧作品卡片 — 视频作品（已完成）：**

| 元素 | 说明 |
|------|------|
| 布局 | 横向布局：左侧视频缩略图(288px) + 右侧信息区 |
| 视频缩略图 | 16:9 比例，中央播放按钮，hover 微放大 |
| 参数标签 | 类型(视频)、模型、比例、分辨率、时长 |
| 提交时间 | 格式 `YYYY-MM-DD HH:mm` |
| 操作按钮 | `重新编辑` / `再次生成` / `收藏为我的素材` |

**右侧作品卡片 — 生成中状态：**

| 元素 | 说明 |
|------|------|
| 进度环 | 48×48 SVG 环形进度（cyan 色） |
| 文字 | 提示词 + 进度百分比 + 参数摘要 |
| 提交时间 | 显示 |
| 动画 | `skeleton-card` 脉冲效果 |

**操作按钮行为：**

| 按钮 | 行为 |
|------|------|
| 重新编辑 | 将该任务的所有参数（类别、提示词、模型、比例、分辨率、时长）回填到输入面板 |
| 再次生成 | 以相同参数创建新任务加入队列，自动开始生成 |
| 收藏为我的素材 | 占位功能（暂未实现） |

### 1.6 ActionDock 悬浮进度栏

| 元素 | 说明 |
|------|------|
| 位置 | 固定底部居中，`bottom: 24px` |
| 样式 | 毛玻璃背景，圆角 24px，暗色半透明 |
| 入场动画 | `dock-slide-up` 从底部滑入 |
| 任务图标 | 40×40 圆角缩略图 + 环形进度覆盖，横向可滚动 |
| 状态动画 | 生成中 → 呼吸辉光(`task-breathe`)；完成 → 闪光(`task-complete-flash`)；入场 → 缩放滑入(`task-slide-in`) |
| hover 悬浮卡 | 向上弹出 Popover：任务名 + 参数标签 + 进度/状态 + 预估时间 |
| 点击任务图标 | 将该任务参数回填到输入面板 |
| 自动隐藏 | **所有任务完成后 3 秒自动消失**；有新任务时重新显示 |
